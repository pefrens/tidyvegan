# GitHub Skills

Skills for GitHub pull request workflows — creating PRs, addressing review feedback, and resolving threads.

## Skills

- **[pr-create](./pr-create/)** - Creates a pull request from current changes, monitors GitHub CI, and debugs any failures until CI passes
- **[pr-threads-address](./pr-threads-address/)** - Review all unresolved PR review threads, address them by making necessary code changes, and commit the changes appropriately
- **[pr-threads-resolve](./pr-threads-resolve/)** - Bulk resolve unresolved PR review threads
