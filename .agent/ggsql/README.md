# ggsql Skills

Skills for writing ggsql queries — a grammar of graphics for SQL.

## Available Skills

### `ggsql`

Write ggsql queries — a declarative SQL extension for data visualization based on Grammar of Graphics principles. Use when the user wants to create, modify, or understand a ggsql visualization query.

**Organization**: Single comprehensive SKILL.md covering query structure (VISUALISE, DRAW, PLACE, SCALE, FACET, PROJECT, LABEL clauses), all layer types, color palettes, common patterns, and CLI usage.

**Resources**:
- [ggsql website](https://ggsql.org)
